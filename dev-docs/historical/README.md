Historical Design Documents

This directory contains archived design explorations, early proposals, and deprecated specifications for Astichi:

- `AstichiApiDesignProposal.md` — original API design proposal.
- `AstichiApiDesignComposed.md` — early composed-form design exploration.
- `AstichiV1Milestones.md` — the original milestone breakdown for V1 implementation. Superseded by the ongoing progress tracking in `../V1ProgressRegister.md` and by the addendum design docs (e.g. `AstichiApiDesignV1-InsertExpression.md`, `AstichiApiDesignV1-UnrollRevision.md`, `AstichiApiDesignV1-BindExternal.md`). Preserved for provenance and original rationale.

These documents are preserved for historical context and to track design rationale, but they do not reflect the current implementation constraints or active API.

Active Specification:
For the normative, up-to-date engineering requirements, please refer to the main AstichiApiDesignV1.md specification in the parent directory.
